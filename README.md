# University Student Management System

## Description
This is a  Python project . It is a command-line based Student Management System. It helps the user to manage student records in easy and organized way.

## Features
- Add Student
- Search Student
- Update Student
- Delete Student
- Show All Students
- Exit Program


## Technologies Used
- Python ,Google Colab

## How to Run
1. Open the project in Google Colab.
2. Run all code cells.
3. Select a menu option.
4. Enter the required information.
